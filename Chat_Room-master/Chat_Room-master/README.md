
# **Real-Time Chat Application**  

This real-time chat application is designed to provide seamless and instant communication between multiple users, featuring a modern, responsive interface and a powerful backend. Built using a combination of cutting-edge web technologies and robust server-side frameworks, the application ensures a smooth and interactive user experience.  

## **Technologies Used:**  
- **Frontend:**  
  - **HTML:** Structure and layout of the web pages.  
  - **CSS:** Styling and design for an attractive and responsive interface.  

  - **Bootstrap:** Ensures a sleek, mobile-friendly, and consistent design across devices.  

- **Backend:**  
  - **Java:** Core programming language for backend logic and processing.  
  - **Spring Boot:** Provides a robust, scalable, and efficient framework for server-side operations.  

- **Real-Time Communication:**  
  - **WebSockets:** Enables instant two-way communication between client and server.  
  - **SockJS & STOMP Protocols:** Ensures reliable WebSocket connections and message handling.  

- **Database:**  
  - **MySQL:** Stores user data and chat messages efficiently and securely.  

- **Deployment & Containerization:**  
  - **Docker:** Packages the entire application into a portable and scalable container.  
  - **Docker Hub:** Stores the Docker image for easy deployment and sharing.  
  - **Render:** Live hosting platform where the application is deployed and accessible online.  

## **Key Features:**  
- **Real-Time Messaging:** Instant chat functionality with real-time message delivery.  
- **Modern UI Design:** Built with Bootstrap to ensure a clean, intuitive, and responsive interface on all devices.  
- **Dynamic Chat Window:** Automatically scrolls to the most recent message for a seamless experience.  
- **Unique Message Animations:** Adds engaging visual effects to incoming and outgoing messages.  
- **User Authentication:** Secure login and registration system with password encryption using Spring Security‚Äôs BCrypt.  
- **Dockerized Deployment:** Streamlines the deployment process, making the application easily scalable and portable.  
- **Live Hosting:** The application is hosted on **Render**, providing real-time access and ensuring uptime and performance.  

## **How It Works:**  
1. **User Registration & Login:** New users can register, and existing users can log in securely.  
2. **Real-Time Chat:** Users can join the chat room and send messages that appear instantly for all participants.  
3. **Message Display:** Sent messages appear on the right, and received messages appear on the left with smooth animations.  
4. **Responsive Design:** The chat interface adjusts perfectly across devices like phones, tablets, and laptops.  
5. **Logout:** Users can securely log out, redirecting them to the home page.
